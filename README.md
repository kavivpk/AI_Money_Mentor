# AI Money Mentor – Data Analytics Based Financial Advisor

A production-ready financial advisor web application with a premium React frontend and a Python Flask backend.

## 🚀 How to Run the Project

Follow these steps to start the application on your computer:

### 1️⃣ Start the Backend (Flask API)
Open a new terminal and run:
```bash
cd backend
pip install -r requirements.txt
python app.py
```
*The backend will run on: **http://localhost:5000***

---

### 2️⃣ Start the Frontend (React UI)
Open another terminal and run:
```bash
cd frontend
npm install
npm start
```
*The frontend will run on: **http://localhost:3000***

---

## 🛠️ Tech Stack
- **Frontend:** React.js, Recharts, CSS3 (Glassmorphism)
- **Backend:** Python, Flask, Flask-CORS
- **Analytics:** Custom Rule-based Financial Engine

## 📊 Core Features
- Financial Health Scoring (0–100)
- Expense-to-Savings Ratio Analysis
- Monthly Income vs Spending Charts
- AI-Powered Personalized Financial Advice
- Age-based Investment Roadmap (Low/Med/High Risk)

---
*Created for Hackathon 2026 — No data is stored, processing is entirely local.*
